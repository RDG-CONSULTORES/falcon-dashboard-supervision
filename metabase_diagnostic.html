🔧 Diagnóstico Metabase - Embedding

⚠️ Problema Identificado

"rdg-consultores.metabaseapp.com refused to connect"

Este error indica que Metabase está bloqueando el embedding del dashboard en iframes.

📊 URL de tu Dashboard:

URL: https://rdg-consultores.metabaseapp.com/public/dashboard/647f87a1-b51d-494e-a5d0-f20ddf100e68

🧪 Test de Iframe (Puede Fallar):

⏳ Haz clic en "Probar Iframe" para comenzar

🚨 Posibles Causas del Error:

  1. Dashboard No Público: El dashboard no está configurado como público
  2. Embedding Deshabilitado: Metabase no permite embedding de este dashboard
  3. Configuración X-Frame-Options: Metabase bloquea iframes por seguridad
  4. CSP Restrictivo: Content Security Policy muy estricto

✅ Soluciones Disponibles:

  1. Usar Enlace Directo: Abrir el dashboard en nueva pestaña
  2. Configurar Embedding: Habilitar embedding en configuración de Metabase
  3. API de Metabase: Usar la API para obtener datos y crear dashboard propio
  4. Dashboard JavaScript: Volver al dashboard con JavaScript que funcionaba

🔧 Pasos para Solucionar en Metabase:

  1. Abrir Metabase Admin Panel
  2. Ir a Settings > Embedding
  3. Habilitar "Enable embedding"
  4. En el dashboard, ir a Sharing & Embedding
  5. Activar "Enable embedding for this dashboard"
  6. Copiar el embedding URL o iframe code
